# Funcionalidade: Persistência das notificações

Toda vez que forem adicionadas novas notificações ao software, elas deverão ser salvas no arquivo `%%appdata%/notifications.json` contendo um array de todas as notificações. Ao iniciar o software deverá carregar as informações salvas no arquivo local e caso excluir uma ou todas as notificações deverá persistir os dados no arquivo também.
